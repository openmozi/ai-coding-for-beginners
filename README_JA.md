## コース概要

近年、大規模言語モデル（LLM）はソフトウェア開発に革命的な新しいパラダイムをもたらしました。従来のソフトウェア開発ライフサイクルは、あらゆる段階でAI自動化によって変革されつつあり、次世代のソフトウェアエンジニアはどのように働くべきかという疑問が生じています。

本コースでは、モダンなAIツールがソフトウェア開発をどのように変革しているかを示します。アプリ開発の敷居を下げる一方で、複雑なアプリケーションには依然としてエンジニアの判断が必要です。本コースでは、従来のソフトウェアエンジニアリングの課題の背後にある理論と、今日それらの問題を解決する最先端のAIツールの両方を扱います。

> **重要**: これは **"vibe coding" コースではありません**。AIの出力を盲目的に信頼することはできません。**Human-agent engineering** がコアです。人間のエンジニアはAIのマネージャーおよび意思決定者として、大量のコードを読んでレビューし、優れたソフトウェアとは何かを理解する必要があります。
